# vázlatok

Ez a repozitórium a vázlatok és képek tárolására szolgál, amelyeket más AI eszközökhöz használhatsz fel.

## Hogyan használd?

1. **Képek feltöltése:**
   - Töltsd fel a képeidet az `images/` mappába.
   - Ezt megteheted a GitHub webes felületén keresztül is ("Add file" -> "Upload files").

2. **Közvetlen link elérése:**
   - A feltöltött kép közvetlen (raw) linkjét így kaphatod meg:
     `https://raw.githubusercontent.com/silentreaver/v-zlatok/main/images/KEP_NEVE.kiterjesztes`
   - Például, ha feltöltesz egy `vazlat1.png` fájlt, a linkje ez lesz:
     `https://raw.githubusercontent.com/silentreaver/v-zlatok/main/images/vazlat1.png`

3. **Használat más AI-okkal:**
   - Ezt a linket másolhatod be Midjourney-be, ChatGPT-be vagy bármilyen más eszközbe, ami kép-URL-t kér.
